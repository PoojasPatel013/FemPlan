## Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/PoojasPatel013/femplan.git
   cd femplan
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the Flask app:
   ```bash
   python app.py
   ```
4. Open the app in your browser:
   ```
   http://127.0.0.1:5000/
   ```
